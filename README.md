# OpenAI Whisper Voice Chat

A Python project that enables voice conversations with ChatGPT using speech recognition and artificial intelligence.  
This application captures audio input, converts speech to text using OpenAI Whisper, sends the message to ChatGPT, and returns intelligent responses.

## 🚀 Features

- Voice input with speech recognition
- Speech-to-text transcription using OpenAI Whisper
- AI-generated responses with ChatGPT
- Simple and interactive Python workflow
- Great project for learning AI integrations

## 🛠️ Technologies Used

- **Python**  
- **OpenAI API**  
- **Whisper (Speech Recognition Model)**  
- **ChatGPT API**  
- **Google Colab / Jupyter Notebook**  
- **Git & GitHub**
